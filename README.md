# Erika Alban - Pool Player Website

Personal website for competitive pool player Erika Alban.

## Setup

This is a static website that can be hosted on GitHub Pages.

## GitHub Pages Setup

1. Push this repository to GitHub
2. Go to Settings > Pages in your GitHub repository
3. Select the branch (usually `main` or `master`) and folder (`/ (root)`)
4. Your site will be available at `https://[username].github.io/[repository-name]`

## Custom Domain

If you want to use `erikaalban.com`:

1. Add a `CNAME` file with your domain name
2. Configure DNS settings with your domain provider
3. Enable custom domain in GitHub Pages settings
